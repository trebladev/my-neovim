# My-neovim-settings

## File structure
```
├── README.md
├── init.vim
├── lua
│   ├── basic.lua
│   ├── colorscheme.lua
│   ├── keybindings.lua
│   ├── plugin-config
│   │   ├── bufferline.lua
│   │   ├── lualine.lua
│   │   └── nvim-tree.lua
│   └── plugins.lua
└── plugin
    └── packer_compiled.lua
```
This file system use init.vim as the entry of nvim settings.

Most plugins' install command was writing in plugins.lua.

Basic and plugins keybinding was writing in keybinding.lua.
